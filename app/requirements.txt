# =============================================================================
# WINDOWS-SPECIFIC NOTES
# =============================================================================
# 1. uvicorn[standard] installs uvloop only on Linux/Mac.
#    On Windows, uvicorn uses asyncio's ProactorEventLoop — no action needed.
#
# 2. psycopg[binary] bundles libpq.dll — no system PostgreSQL client required.
#
# 3. Celery on Windows MUST use solo pool:
#    celery -A app.tasks worker --pool=solo --loglevel=info
#    CELERY_WORKER_CONCURRENCY=1 enforces this via config.
#
# 4. Selenium: webdriver-manager auto-downloads ChromeDriver for Brave/Chrome.
#    If auto-detection fails, set BROWSER_BRAVE_PATH in .env.
#
# 5. Python 3.14 ABI note:
#    This requirements.txt targets the STANDARD cp314 build (GIL enabled),
#    installed by default from python.org. The free-threaded build (cp314t,
#    --disable-gil) is a separate optional installer — do not mix the two.
